Railway 배포된 API의 두 가지 버그를 수정하고, 신규 배포 후 라이브 환경에서 QA 검증을 완료한다.

## 발견된 버그

### Bug 1: `POST /ai/translate-explain` → 500 에러 (Critical)
- **원인**: `AiService.translateExplain()`에서 Anthropic SDK 호출(`this.client.messages.create()`)에 try/catch 없음
- Anthropic 크레딧 소진 시 SDK가 `BadRequestError` (400) throw → NestJS가 처리 못해 500 반환
- **증상**: 프론트엔드에서 번역 요청 시 서버 크래시

### Bug 2: `GET /dictionary/search` → `[]` 빈 배열
- **원인**: 이전 세션에서 DB에 "preparation" 1개만 있던 상태에서 `seedDictionaryIfEmpty()`의 조건이 `count > 0`이었으므로 시드 스킵됨
- Railway 내부 네트워크(`postgres.railway.internal`)는 외부에서 직접 접근 불가 → 로컬에서 seed.ts 실행 불가
- **증상**: 사전 검색 기능 전체 미동작

---

## 수정 내용

### Fix 1: AI 번역 에러 핸들링 (`apps/api/src/modules/ai/ai.service.ts`)

`translateExplain()` 내 Anthropic API 호출을 try/catch로 감싸 503 반환:

```typescript
let message: Awaited<ReturnType<typeof this.client.messages.create>>;
try {
  message = await this.client.messages.create({
    model: 'claude-haiku-4-5-20251001',
    max_tokens: 512,
    messages: [{ role: 'user', content: prompt }],
  });
} catch {
  throw new HttpException(
    'AI translation service is temporarily unavailable. Please try again later.',
    HttpStatus.SERVICE_UNAVAILABLE,
  );
}
```

### Fix 2: 사전 시드 임계값 수정 (`apps/api/src/prisma/prisma.service.ts`)

```typescript
// Before
if (count > 0) return;

// After
if (count >= 50) return;
```

- DB에 50개 미만의 항목이 있으면 시드 실행 (기존 1개짜리 잔존 데이터 대응)
- `createMany({ skipDuplicates: true })`와 함께 사용하여 중복 방지는 `@id` 기준으로 작동

### Bug 3: AI 설명 버튼 401 Unauthorized (2026-03-29)
- **원인**: Access token 만료 시간이 `15m`으로 짧아, 레슨 학습 중 토큰 만료 시 AI 엔드포인트 401 에러 발생
- **증상**: 카드 플립 후 "AI 설명 보기" 클릭 시 401 반환, 프론트엔드에서 `common.ai_error` 표시
- **Fix**: `apps/api/src/modules/auth/auth.service.ts` — `expiresIn: '15m'` → `'1h'`
- **커밋**: `f4df2ec`

### Bug 4: AI Explain 503 — Anthropic 크레딧 소진 (2026-04-10)
- **원인**: `ANTHROPIC_API_KEY`는 설정됐으나 Anthropic 계정 크레딧 잔액 0 → API 호출 시 `invalid_request_error: credit balance too low` 반환
- **증상**: "AI 설명 보기" 클릭 시 503 반환, 모든 AI 기능 불동작
- **진단 과정**: debug 응답 노출 커밋 배포 → `"Your credit balance is too low to access the Anthropic API"` 확인
- **Fix**: Anthropic API 호출 제거 → **사전 DB 기반 정적 설명 생성**으로 교체
  - `ai.service.ts`: `DictionaryEntry` 조회 + 태그(hanja/loanword/greeting 등) 기반 설명 빌더
  - `translate.dto.ts`: `cardKo/cardJa/cardKoReading/cardJaReading` 옵셔널 필드 추가
  - `lesson/page.tsx`: 카드 전체 데이터를 API에 전송 (`handleAiExplain(card)`)
  - 일일 한도(DAILY_LIMIT 20) 제거 → 무제한 (`remainingToday: 999`)
  - `generateTip()` 정리 (대시보드 이미 정적 팁, T-0028)
- **커밋**: `fcae0ea`
