## 작업 내용

### 1. apps/web/lib/tips.ts 생성

500개의 트릴링구얼(EN/KO/JA) Tip 객체 배열 생성:

```typescript
export interface Tip {
  en: string;
  ko: string;
  ja: string;
}
export const TIPS: Tip[] = [ /* 500개 항목 */ ];
```

#### 주제 분류 (50개 단위)

| 범위 | 주제 |
|------|------|
| 1-50 | 핵심 교육/학교 한자어 (학교/學校, 도서관/圖書館 등) |
| 51-100 | 의료/건강 한자어 및 외래어 |
| 101-150 | 직장/사회 한자어 |
| 151-200 | 동일 외국어 출처 차용어 (포르투갈어, 네덜란드어, 프랑스어 등) |
| 201-250 | 문법 평행 구조 (SOV, 조사 체계, 경어 등) |
| 251-300 | 요일/시간 표현 (월화수목금토일 공유) |
| 301-350 | 문화적 공통점 (음식, 명절, 목욕 문화 등) |
| 351-400 | 발음/음운론 사실 |
| 401-450 | 같거나 비슷한 어휘 (의미 일치 확인) |
| 451-500 | 문자 체계 역사 및 한중일 문화 연결 |
