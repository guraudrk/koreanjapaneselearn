## 1. 프로젝트 개요

- **프로젝트명**: 전 세계 사람들이 한국어와 일본어를 기초부터 쉽게, 동시에 학습할 수 있는 플랫폼 (웹 및 앱)
- **목적**: 한국어와 일본어에 동시에 관심을 가진 사람들이 두 언어를 함께 배울 수 있도록 돕는 언어 학습 플랫폼(웹/앱) 제작
- **해결 문제**: 한국/일본에 대한 관심이 높아지면서 두 언어를 동시에 배우고자 하는 수요가 증가했으나, 이를 동시에 지원하는 서비스가 없음. 두 언어의 공통점/차이점을 상세히 설명해주는 서비스도 부재.

---

## 2. 제품 비전

- **비전**: 전 세계 누구나 한국어와 일본어를 기초부터 쉽게, 동시에 학습할 수 있는 최고의 플랫폼
- **핵심 가치**: 사용자 중심의 직관적이고 효율적인 이중 언어 학습 경험

---

## 3. 타겟 사용자

- 전 세계의 한국어와 일본어를 동시에 배우고 싶은 사람들
- 한국어 단독 학습자
- 일본어 단독 학습자

---

## 4. 핵심 기능 (4가지)

### 기능 1 — 언어 학습 (기초~고급)
- 한국어/일본어를 동시에 배우는 학습 모드
- 선택 기능: 한국어만 / 일본어만 / 동시 학습 전환 스위치
- 비교 학습 카드: 동일 한자어(예: 준비-準備) 및 문법 구조를 동시에 표시
- 단계별 커리큘럼: 히라가나/한글 기초 → 비즈니스 회화까지 레벨링

### 기능 2 — AI 번역/설명 모델
- 사용자 모국어 입력 → 한국어/일본어 번역 + 문법/문화 뉘앙스 설명
- 일일 사용 제한 (무료 티어)
- 유료 구독 시 무제한 + 고성능 AI 사용 가능
- 다국어 음성 인식(STT): 모국어 음성 → KR/JP 변환 (Post-MVP)
- 심층 설명 AI: 번역 문장의 문법적 특징, 존댓말 등 문화적 뉘앙스 설명

### 기능 3 — 놀이 기능
- 자신의 사진을 애니 캐릭터 또는 한국풍 캐릭터로 변환하는 AI 기능
- 변환 결과 갤러리/공유 (Post-MVP)

### 기능 4 — 공유/성장 루프
- 학습 시마다 포인트 적립
- 포인트 및 학습 성과를 친구와 공유
- 학습 카드/성과 공유 링크 생성 (OG 이미지 포함)

---

## 5. 부가 기능

- 사용자 친화적 인터페이스
- 반응형 디자인 (웹/모바일)
- 데이터 보안 및 프라이버시 보호
- 백과사전: 영어로 검색 시 한국어/일본어 뜻 반환
- 대시보드: 오늘 학습 현황, 진도 시각화
- 알림 시스템
- 구독 티어 관리

---

## 6. 비즈니스 요구사항

- **목표**: 월 매출 1,000만원 달성, 브랜드 인지도 제고, 사용자 기반 확대
- **성공 지표**: 사용자 만족도, 월간 활성 사용자(MAU), 서비스 안정성
- **수익 모델**: 무료(일일 AI 제한) + 유료 구독(무제한/고성능)

---

## 7. 기술 요구사항

- 최신 웹 기술 활용 (Next.js, NestJS, React Native/Expo)
- 빠른 응답 속도 및 안정적인 서비스 (99% 이상 가용성)
- 데이터 암호화 및 접근 제어

---

## 8. MVP 범위 (우선순위 고정)

1. 회원가입/로그인
2. 학습 모드 선택: KR / JP / BOTH
3. 단계별 커리큘럼 + 학습 카드/퀴즈 + 진도 저장
4. 백과사전 검색 (영어 입력 → KR/JP 뜻 + 예문)
5. 포인트 적립 + 기본 공유
6. AI 기능 1종(텍스트): "내 언어 → KR/JP 번역 + 간단 설명" + 일일 제한

Post-MVP: STT, 구독/결제, 이미지 변환 등 (MVP 완료 후)
